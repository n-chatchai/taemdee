# แต้มดี · 30-Day Solo Launch Plan

*Created: 2026-05-07*
*Constraints: solo · 30-60 min/วัน · ไม่เดินสนาม · Thailand-wide · FB+TikTok BiP*
*Goal: 20-30 ร้านกาแฟ indie sign up · 5+ testimonials · 500+ FB followers*

## Strategy
**Two channels feeding each other:**
- **#1 Cold DM (IG)** = active acquisition · 20-30 ร้าน sign up
- **#2 Build-in-Public (FB)** = passive trust + audience compound
- TikTok เลื่อนไปเดือน 2 (รอ testimonial footage)

**Loop:** DM กับร้าน → ร้าน sign up → progress + screenshot → FB post → audience grows → DM response rate ขึ้น

---

## Week 1 · Setup (no acquisition yet)

### วัน 1-2 · Asset prep (60 min/day)
- [ ] **Domain + landing page live** ที่ taemdee.app (หรือ subdomain)
  - Hero: "บัตรแต้มที่ลูกค้าไม่ทำหาย · ไม่ต้องโหลดแอป · ไม่จ่ายรายเดือน"
  - 1 CTA: "เปิดร้านฟรี" → onboarding
  - Manifesto + 3-5 objection answers จาก context
- [ ] **FB Page "แต้มดี"** สร้าง · cover + bio + first post (intro)
- [ ] **IG account** "@taemdee.app" (หรือ name available) · bio + link
- [ ] **Spreadsheet "DM Pipeline"** (Google Sheet) — columns: shop_name, IG_handle, location, follower_count, niche (specialty/regular/dessert), DM_date, response, status (cold/replied/interested/signed_up/declined), notes

### วัน 3-4 · Build target list (60 min/day)
- [ ] **Scrape 200 IG accounts** ของร้านกาแฟ indie ไทย · 1k-20k followers
  - Search hashtags: `#ร้านกาแฟ` `#ร้านกาแฟไทย` `#specialtycoffeethailand` `#cafehopthailand` `#กาแฟไทย` `#cafebkk` `#cafechiangmai`
  - Criteria: indie (ไม่ใช่ chain) · มีหน้าร้านจริง · post ใหม่ใน 30 วัน · ตอบ comment
- [ ] **Filter ลงเหลือ 100** ที่ดูเป็น "owner-operator" (เห็นในรูป + caption เป็น 1st person)

### วัน 5 · Content prep (60 min)
- [ ] **DM template 3 variants** ใน sheet (A/B/C)
- [ ] **5 FB post drafts** (templates ด้านล่าง) ใน Notes/Google Docs
- [ ] **Screenshots clean จาก dashboard + customer flow** เก็บไว้ในโฟลเดอร์ — ใช้ใน FB post

### วัน 6-7 · Soft launch (30 min/day)
- [ ] **First FB post: "วันนี้เริ่มแล้ว"** — open the kimono · ผมทำแอพให้ร้าน SME ไทย
- [ ] **Pin landing link** ที่ FB Page + IG bio
- [ ] **Test DM ด้วยตัวเอง** ส่งให้ตัวเอง/เพื่อน 5 DMs ทดสอบ flow + response

## Week 2-4 · Execution (30-60 min/day)

### Daily routine (60 min)
**Morning batch (30 min):**
- DM 25-30 ร้าน (IG) · ใช้ template + personalize 1 line ต่อ DM
- Update sheet · มาร์ค sent

**Evening batch (30 min):**
- ตอบ DM replies + ตอบใน inbox (priority #1)
- โพสต์ FB 1 post (ใช้ template ด้านล่าง · 5 นาที edit)
- Comment ใน FB groups 2-3 ที่ relevant (ไม่ขายตรง)

### Weekly cadence
- **จันทร์-ศุกร์:** DM + FB post
- **เสาร์:** review week metrics + plan content next week
- **อาทิตย์:** rest หรือ batch FB content drafts

### Weekly review template
- DM sent · DM replied · % reply
- Sign ups · % sign up จาก reply
- FB followers · FB engagement
- Best post · worst post
- Insight ของสัปดาห์

## #1 Cold DM Playbook

### Target: IG indie café Thailand
- 1k-20k followers (sweet spot — owner-operator, ตอบ DM เอง)
- Active (post ใน 30 วัน)
- Indie (ไม่ใช่ chain)

### DM template (เริ่มที่ 3 variants A/B/C)

**Variant A (warm/personal):**
```
สวัสดีครับ พี่[ชื่อร้าน]
ผมเห็น [กาแฟตัว / ร้าน / shot signature] ของพี่ใน feed บ่อยๆ
ผมทำของชื่อ "แต้มดี" เป็นบัตรสะสมแต้มดิจิทัลให้ร้าน SME · ไม่ต้องโหลดแอป
ขอให้พี่ใช้ฟรี + อยากได้ feedback จากพี่ครับ · ใช้เวลาตั้งร้านแค่ 60 วิ
สนใจมั้ยครับ?
```

**Variant B (problem-led):**
```
สวัสดีครับ
สงสัยว่าพี่[ชื่อร้าน]ทำบัตรสะสมแต้มแบบไหนอยู่ครับ?
ผมทำบัตรแต้มดิจิทัลที่ลูกค้าไม่ต้องโหลดแอป · ฟรีตลอด
ขอให้พี่ลองใช้ + เก็บ feedback ครับ
ถ้าสนใจส่ง [link landing] กลับไปได้
```

**Variant C (curiosity):**
```
สวัสดีครับ พี่[ชื่อร้าน] · ผมหา beta tester 30 ร้านสำหรับแอพบัตรสะสมแต้มที่ลูกค้าไม่ต้องโหลด
ฟรีตลอด · ตั้งใน 60 วิ
ถ้าสนใจตอบกลับมาได้ครับ
```

**A/B test:** ส่ง variant A: 50 ร้าน · B: 50 · C: 50 ใน Week 2 · เลือกตัวที่ reply rate สูงสุดใน Week 3

### Reply handling
- **สนใจ:** ส่ง landing link + offer ช่วย setup ผ่าน screenshot
- **ถามต่อ:** ตอบสั้น + reference objection answer จาก context
- **เงียบหลัง 5 วัน:** follow-up 1 ครั้งเดียว (sale manual: "ทุก 'ไม่' วันนี้คือ 'ยัง'")
- **ปฏิเสธ:** ขอบคุณ · ทิ้ง landing · กลับไปอีกที 30 วัน (ไม่ฝืน)

### Daily target
- **30 DMs/วัน** = 600/เดือน
- Target reply rate: 15% = 90 replies
- Target sign-up rate (จาก replies): 25% = 22 ร้าน · เกิน goal

## #2 FB Build-in-Public Playbook

### Voice: founder talking to peers
- 1st person · "ผม"
- เปิดเผย metric · failure · learning
- ไม่ขายของตรง · เล่ายากกว่า

### Content pillars (5 ประเภท · rotate)

**A. "วันนี้ลงสนาม" — daily progress**
- "วันที่ 14 · DM 28 ร้าน · ตอบกลับ 5 · sign up 1"
- "วันที่ 21 · ลูกค้าร้านแรกใช้ครบ 10 แก้ว ฟรีจริง"

**B. "เจ้าของร้านพูดอะไร" — verbatim feedback (anonymized)**
- "พี่ร้านนึงบอกผม: 'ฟรีจริงเหรอ? ของอะไรมาหลอกอีก'"
- ตอบ + insight สำหรับ founder คนอื่น

**C. "ทำไมแต้มดี" — manifesto bites**
- 1 ของ 4 pillars (REIMAGINE/EASY/AFFORDABLE/OPEN) ต่อโพสต์
- ใช้ "BY SME · FOR SME pledge"

**D. "เบื้องหลัง" — process / learning**
- "ผมแก้ onboarding 3 รอบเพราะลูกค้าร้านแรก feedback ว่า..."
- "ผม design เอง · ใช้ time แต่ละหน้าน้อย < 60 วินาที"

**E. "Tip ร้าน SME" — value-first content (ไม่ขายของ)**
- "3 วิธีให้ลูกค้ากาแฟกลับมาซ้ำ (ไม่ต้องลดราคา)"
- "ทำไมบัตรกระดาษถึงไม่เวิร์คอีกแล้ว"

### Posting cadence
- **5 posts/สัปดาห์** (จ-ศ) · 1 post/วัน
- Mix: 2× A/B (progress/feedback) · 2× E (value tip) · 1× C/D (manifesto/BTS)

### Hook templates (เริ่ม post)
- "วันที่ X · [number]"
- "เจ้าของร้านบอกผมว่า..."
- "3 อย่างที่ผมเรียนรู้จากการ DM 100 ร้านกาแฟ"
- "ทำไม[X]ไม่เวิร์คอีกแล้ว"
- "Update: ผมทำ[X]เสร็จแล้ว"

### Engagement rule
- ตอบ comment ทุกอันใน 24 ชม.
- Comment ใน 3 FB groups/วัน (Café-related) · ตอบจริง · ทิ้ง link เฉพาะที่ relevant

### FB groups ที่ควร join
- "ร้านกาแฟ Thailand"
- "เจ้าของร้านกาแฟ"
- "Specialty Coffee Thailand"
- "SME Marketing"
- "Indie Cafe Owners"
- (verify name + active level ใน Week 1)

## TikTok (เลื่อนไปเดือน 2)

**ทำไมเลื่อน:** TikTok ต้องการ video footage จาก customer สำหรับ POV "เจ้าของร้าน" · ต้องรอ testimonial จาก #1 ก่อน · 30-60 min/วัน ไม่พอสำหรับ 3 platforms

**เดือน 2 plan:**
- 1 batch shoot 1 ครั้ง/สัปดาห์ (2 ชม. เสาร์)
- ผลิต 3-5 vid/สัปดาห์ จาก batch
- Repurpose จาก FB content + เจ้าของร้าน testimonial
- Format: 30-60 วิ · talking head + screen recording

## Risks + mitigations

- **DM ban (IG)** → จำกัด 30 DMs/วัน · ไม่ copy-paste exact same · personalize 1 line
- **No reply** → ไม่เปลี่ยน strategy ก่อน sample size 100 DM/variant · A/B aware
- **TikTok bottleneck** → ยอมรับเลื่อนเดือน 2 · ไม่ฝืน
- **Burnout 30-60 min/วัน sustained** → take Sunday off · batch content วันเสาร์
- **Thailand-wide ไม่ได้ word of mouth** → mitigation = #2 carry weight (founder presence + content)

## Out of scope (อย่าทำใน 30 วัน)
- LINE OA marketing
- SEO content engine (ทำเดือน 2-3)
- Free tool (เดือน 3+)
- Paid ads (รอจน product-market signal ก่อน)
- Influencer collab (เดือน 2+ · รอ testimonial)
- Email (ไม่ใช่ Thai SME channel)
- Product Hunt / SaaS directories (เดือน 2-3 · บัง backlink)
