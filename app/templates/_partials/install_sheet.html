{# Install sheet — auto-pops in regular browser tabs (not the installed PWA). One sheet, both instruction sets shown side by side. We don't try to UA-detect the platform anymore: Chrome on iOS, Firefox on Android, less common Edge/Brave/etc all blur the lines, and getting one user the "wrong" instructions is worse than showing both neutrally. The Android-style native install CTA only appears when `beforeinstallprompt` actually fires (Chromium-based browsers on Android), so platforms that can't trigger it skip the button and read the instructions instead. #}
